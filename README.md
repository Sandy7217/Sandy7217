> **RoboRana AI** is a **local-first, privacy-preserving multi-agent AI system** purpose-built for e-commerce analytics and automation. It runs entirely on your machine — no cloud dependency, no data leaks.

### 🏗️ Architecture

```

│   │  Sales   │  │Inventory │  │ Finance  │  │   Ads    │  │
│   │  Agent   │  │  Agent   │  │  Agent   │  │  Agent   │  │
│   └────┬─────┘  └────┬─────┘  └────┬─────┘  └────┬─────┘  │

│   │ ChromaDB  │    │  LangChain   │    │    n8n    │       │
```

### ⚡ Key Features

| Feature | Description |
|---------|-------------|
| 🤖 Multi-Agent Architecture | Specialized AI agents for Sales, Inventory, Finance, Ads |
| 🧠 RAG Knowledge Retrieval | ChromaDB vector store + LangChain for context-aware responses |
| 📊 Power BI Integration | Real-time e-commerce dashboards with AI-generated insights |
| 🔄 n8n Automation | Visual workflow automation for repetitive analytics tasks |
| 🏠 Local-First | Runs on Ollama — 100% private, no cloud dependency |
| 🎭 Adaptive AI Personality | Dynamic agent personality system for better UX |
| 📈 Forecasting Engine | ML-powered sales and inventory forecasting |
